An interface of the Envoy Gauge.

A Gauge can be both incremented and decremented.

Examples

Basic usage of Gauge:

use envoy::host::Stats;

let stats = Stats::default();

let requests_active = stats.gauge("requests_active")?;

requests_active.inc()?;

... do some work ...

requests_active.dec()?;
